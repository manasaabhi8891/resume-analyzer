# 🤖 AI Resume Analyzer

An AI-powered web application that analyzes resumes against job descriptions and provides insights to improve ATS (Applicant Tracking System) score.

## 🚀 Features

- 📄 Upload Resume (PDF)
- 🧾 Analyze Job Description
- 📊 Match Score Calculation
- ❌ Missing Skills Detection
- 🤖 AI-Based Suggestions
- ⭐ ATS Score Breakdown
  - Skill Match
  - Keyword Match
  - Resume Quality

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- PyPDF2
- spaCy
- HTML, CSS

## 📸 How It Works

1. Upload your resume (PDF format)
2. Paste the job description
3. Click "Analyze"
4. Get:
   - Match Score
   - Missing Skills
   - AI Suggestions
   - ATS Score Breakdown

## 🌐 Deployment

Deployed using Render.
